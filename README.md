# SimpleNav
Remove unwanted DOM elements from specific websites using custom CSS selectors

Take control of your browsing experience.

Are you tired of massive cookie banners taking up half the screen? Frustrated by "Subscribe Now" popups that won't close? Or websites that disable your scroll bar?

SimpleNav is a Chrome/Firefox extension that will become the ultimate power tool for users who want a cleaner, faster, and distraction-free web. Unlike standard ad-blockers that work on general lists, SimpleNav lets you decide exactly what stays and what goes.

With a simple and organized interface, you can create custom rules to remove elements, modify styles, or manage CSS classes on any domain.
